### css 權重

1. inline style : 1, 0, 0, 0
2. ID 選擇器 : 0,1,0,0
3. class 選擇器、屬性選擇器、偽類 : 0, 0, 1, 0
4. 元素標籤、偽元素 : 0, 0, 0, 1

連接符號，例如: 空白(後代)、>(子代)、+(相鄰)、~(全體)、//、^、$、\*、:not() 對權重沒有影響
但:not() 括號內部的參數會算分數
