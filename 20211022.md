- 圖形內用文字工具點 -> 區域文字

### ICMP (網際網路控制訊息協定 Internet Control Message Protocol)

- 功能: 錯誤報告、診斷功能
- 確認 IP 間的資料傳輸是否正確運作
- 透過 ping 指令確認 IP 位址之間的通訊是否正常

### ARP (位址分析協定 Address Resolution Protocol)

- 取得與接收端 IP 位址相對應的 MAC 位址，此稱為"位址解析"
