### vue data

- Vue 會自動將 data 內的屬性加上 getter 與 setter 的特性，以便監控狀態的更新
- Vue 實體被建立後，Vue 會自動為這個實體加上 $data 屬性，可以透過 vm.$data.XXX 來操作內部狀態
  - 如果我們沒有將 Vue 實體掛載至某個 DOM 節點，賦值時會報錯

```javascript
const vm = new Vue({
  data() {
    return {
      name: 'wow!'
    };
  }
}).mount('#app');

vm.$data.name = 'Hello Vue!';
```
